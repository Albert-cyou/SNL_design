## 词法分析

文件：`make_token.py`
定义了`token`的数据结构
```     
class Token:
    def __init__(self, Lineshow, Lex, Sem):
        self.Lineshow = Lineshow  # 行号
        self.Le x = Lex  # 单词的词法信息 t,;
        self.Sem = Sem  # 单词的语义信息 ID,SEMICOLON...
```
主要功能：把代码变成tokens序列，需要准确识别关键字类型，保存在`Sem`中（如"`ID`"表示这个token是个参数，"`IF`表示这个是if关键字"）；`Lex`保存token的类型，比如是`ID`,名字是`t`

代码没有完全按照状态转移实现，这个自动机可以用if完全实现。不知道有没有问题。

语法树结点包含的信息：
1. 子节点：
2. 节点类型：表示是根节点，程序头，类型声明，还是什么其他的类型。
3. 变量名字：name

语法树结构：
