# 需要用到的 MIPS 指令集合

SNL 当前只处理整数，关系运算只生成 `<` 和 `=`，所以目标代码生成主要使用字运算、整数输入输出、条件跳转和过程跳转。

| NAME | MNEMONIC | OPERATION |
|:--:|:--:|:--|
| Add | `add rd, rs, rt` | `R[rd] = R[rs] + R[rt]` |
| Add Immediate | `addi rt, rs, imm` | `R[rt] = R[rs] + SignExtImm` |
| Add Imm. Unsigned | `addiu rt, rs, imm` | `R[rt] = R[rs] + SignExtImm` |
| Subtract | `sub rd, rs, rt` | `R[rd] = R[rs] - R[rt]` |
| Subtract Unsigned | `subu rd, rs, rt` | `R[rd] = R[rs] - R[rt]` |
| Multiply | `mult rs, rt` | `{Hi, Lo} = R[rs] * R[rt]` |
| Multiply Unsigned | `multu rs, rt` | `{Hi, Lo} = R[rs] * R[rt]` |
| Divide | `div rs, rt` | `Lo = R[rs] / R[rt]; Hi = R[rs] % R[rt]` |
| Divide Unsigned | `divu rs, rt` | `Lo = R[rs] / R[rt]; Hi = R[rs] % R[rt]` |
| Move from Hi | `mfhi rd` | `R[rd] = Hi` |
| Move from Lo | `mflo rd` | `R[rd] = Lo` |
| Branch On Equal | `beq rs, rt, label` | `if R[rs] == R[rt] goto label` |
| Branch On Not Equal | `bne rs, rt, label` | `if R[rs] != R[rt] goto label` |
| Branch Less Than | `blt rs, rt, label` | `if R[rs] < R[rt] goto label` |
| Jump | `j label` | `PC = label` |
| Jump and Link | `jal label` | `$ra = PC + 4; PC = label` |
| Jump Register | `jr rs` | `PC = R[rs]` |
| Load Word | `lw rt, offset(base)` | `R[rt] = M[R[base] + offset]` |
| Store Word | `sw rt, offset(base)` | `M[R[base] + offset] = R[rt]` |
| Load Upper Imm. | `lui rt, imm` | `R[rt] = imm << 16` |
| Load Immediate | `li rt, imm` | 伪指令，装入立即数 |
| Load Address | `la rt, label` | 伪指令，装入数据标签地址 |
| Load Byte | `lb rt, offset(base)` | 读入有符号字节 |
| Load Byte Unsigned | `lbu rt, offset(base)` | 读入无符号字节 |
| Load Halfword | `lh rt, offset(base)` | 读入有符号半字 |
| Load Halfword Unsigned | `lhu rt, offset(base)` | 读入无符号半字 |
| Store Byte | `sb rt, offset(base)` | 写入字节 |
| Store Halfword | `sh rt, offset(base)` | 写入半字 |

## syscall

| SERVICE | `$v0` | ARGS | RESULT |
|:--:|:--:|:--|:--|
| `print_int` | `1` | integer in `$a0` | |
| `print_string` | `4` | string address in `$a0` | |
| `read_int` | `5` | | integer in `$v0` |
| `read_string` | `8` | buffer in `$a0`, length in `$a1` | |
| `exit` | `10` | | terminate program |

## 当前目标代码生成约定

- 所有 SNL 整数按 4 字节 word 存储。
- 符号表暂未分配有效栈偏移，所以生成器使用 `__snl_var_<idx>` 形式的数据标签保存变量、临时变量和形参槽。
- 数组元素地址由 `AADD` 生成，计算方式是 `base + index * 4`；目前不做数组越界检查，也不处理下界偏移。
- `<` 和 `=` 会生成 0/1 布尔值，再由 `JUMP0` 使用 `beq cond, $zero, label` 跳转。
- 过程调用使用 `jal/jr` 和栈上传参；当前适合课程设计里的简单过程调用，未保存嵌套调用时的 `$ra`。
